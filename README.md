# ❤️ Heart Disease Prediction App

A simple Machine Learning web app that predicts whether a person is at **high risk or low risk of heart disease** based on health inputs.

---

## 🚀 What This Project Does

* Takes user health details as input
* Processes the data using a trained ML model
* Predicts:

  * 🔴 High Risk
  * 🟢 Low Risk

---

## 🖥️ Built With

* Python
* Streamlit
* Scikit-learn
* Pandas

---

## 📊 Input Parameters

* Age
* Sex
* Chest Pain Type
* Blood Pressure
* Cholesterol
* Blood Sugar
* ECG Result
* Heart Rate
* Exercise Angina
* Oldpeak
* ST Slope

---

## 📈 Output

* **High Risk of Heart Disease**
* **Low Risk of Heart Disease**
